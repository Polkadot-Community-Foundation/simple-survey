# Security Policy

This repository contains an **experimental proof-of-concept**. It has not been
audited, is actively experimental, and may contain bugs, vulnerabilities, or
incomplete features. It is published for research, experimentation, and
developer education only. Use at your own risk.

## Reporting a vulnerability

If you discover a bug, exploit, or similar vulnerability, you can report it
through this repository's public channels:

- Open a [GitHub issue](../../issues), or
- Use [GitHub's private security advisory](../../security/advisories/new) for
  sensitive reports.

Reports are welcome, but please note that **no fixes, responses, or timelines
are promised**. As a proof-of-concept, this code is provided as-is, and we may
choose not to act on any given report.

## Scope

This policy covers only the code in this repository. It does not cover any
third-party deployment, fork, or downstream operation of this code. Downstream
operators incorporate changes at their own discretion and are responsible for
the security of their own deployments.
